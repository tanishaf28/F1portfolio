# F1 themed Racing Portfolio
A compact, racer-themed portfolio built with Vite, React, and Tailwind. Designed for speed, contrast, and easy customization: Ferrari-inspired color system, neon accents, carbon textures, and optional racer headings.

---
## Tech stack

- React 18
- Vite
- TypeScript
- Tailwind CSS
- Radix UI (components)
- Framer Motion (animations)
- React Router

## Features

- Home / Hero section
- About, Projects, Articles, Photography pages
- Contact form
- Theme toggle (light/dark)
- Reusable UI components in `src/components/ui`
- Mobile-friendly and accessible patterns

## Quick start (local development)
Clone the repo and install dependencies:

```bash
git clone https://github.com/tanishaf28/portfolio.git
cd my-portfolio
npm install
```

Run the development server:
```bash
npm run dev
```
Open http://localhost:5173 (or the port Vite prints) in your browser.


## Project structure (key files)

- `index.html` — base HTML template
- `src/main.tsx` — app entry
- `src/App.tsx` — root app component / routes
- `src/components` — page components (Hero, About, Projects, etc.)
- `src/components/ui` — shared UI primitives and design-system components
- `src/pages` — route-level pages
- `public/` — static assets
- `tailwind.config.ts` — Tailwind configuration
- `vite.config.ts` — Vite configuration
